### Шаг 3: Расширить `dsp_assistant/cli/main.py`

```python
@cli.group()
def rag(): ...

@rag.group()
def meta(): ...

@meta.command("build")
@click.option("--repo", multiple=True)
@click.option("--all", "all_repos", is_flag=True)
@click.option("--dry-run", is_flag=True)
@click.option("--no-embed", is_flag=True)
@click.pass_context
def meta_build(...):
    cfg = load_stack(ctx.obj["stage"])
    db = DbClient(cfg); db.connect()
    extr = MetaExtractor(db, dry_run=dry_run, embed=not no_embed)
    if all_repos: results = extr.build_all()
    elif repo: results = sum([extr.build_repo(r) for r in repo], [])
```

## Ключевые проверки idempotency

- В `_upsert_block`: `SELECT source_hash FROM doc_blocks WHERE block_id = %s`
- Если совпадает → skip, status="skipped", embed НЕ запускаем
- Если отличается → UPDATE + переэмбеддить
- Если NULL → INSERT + embed
