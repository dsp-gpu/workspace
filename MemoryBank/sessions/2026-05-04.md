### 2. Native crashes (5 тестов) ✅

Корневая причина — **отсутствие `py::keep_alive<1, 2>()` на 26 init-конструкторах** в 20 binding-файлах. Locel `ctx` Python переменная уничтожалась после `setUp()`, оставляя dangling reference в C++ объекте → segfault при первом методе. Fix через regex-обход всех `.def(py::init<...GPUContext...>(),`.

Бонус-fix: `py_form_signal_rocm.hpp` всегда возвращает 2D `(antennas, points)` (раньше для `antennas=1` возвращал 1D — inconsistent). После 8 точечных правок в `t_form_signal.py` — все `gen.generate()` обёрнуты в `np.squeeze()`.

Финал: **0 SKIP / 0 FAIL** на gfx1201.
