## 🧠 Уроки сессии

1. **Offline-pack под чужую ОС не работает**: 76 .deb ROCm devkit собранных дома на WSL2 Ubuntu noble **не лезут** на Debian 13 trixie (разная libc, gcc-13 toolchain conflict). Решение — Debian-native apt с noble repo подключённым в sources.list. `--no-install-recommends` ROCm-only без gcc-13 toolchain — путь.

2. **GCC 14.2 + compound namespace**: `using namespace ::drv_gpu_lib;` **внутри** `namespace dsp::X {}` не работает в GCC 14 на Debian 13 (compound nested namespace specifier). Нужно выносить `using` в global scope **перед** namespace declaration.

3. **Fake-nested namespace shadowing**: внутри `namespace dsp::stats { ... namespace dsp::stats::snr_defaults { ... } }` второе объявление создаёт *fake* `dsp::stats::dsp::stats::snr_defaults` (shadow). Использовать **относительные** namespace declarations: `namespace snr_defaults { ... }`.

4. **TestRunner output parsing**: ловить SKIP через `'SKIP' in out.upper()` — **ошибка**, т.к. TestRunner всегда печатает `0 skipped`. Парсить только через regex `Total: N passed, M failed, K skipped` и сравнивать конкретные числа.

5. **GPUWorkLib /home/alex/C++/GPUWorkLib** — рабочий референс legacy кода. При вопросах «как было раньше» — смотреть туда.
